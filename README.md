# This is a Basic Library Control System

### Used Techs
- ReactTS
- TypeScript
- PostgreSQL
- Sequelize
- Tailwind
- Ant Design

## ENV

```bash
DATABASE_URL="<DATABASE URL>"
```

## Getting Started

First, run the development server:

```bash
npm start
# or
yarn start
# or
pnpm start
# or
bun start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
